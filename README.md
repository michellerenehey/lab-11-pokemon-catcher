## user events
* Page Load

  * generate three pokemon 
  * update state -- **pokemonAppear**
  * update view based on state -- render
* Button Click
  * increment total plays state -- **pokemonCaught**
  * increment selected pokemon state -- **pokemonChosen**
  * if (plays < 10) 
      * generate three pokemon 
      * update state -- **pokemonAppear**
      * update view based on state -- render
  * else () 
    * send user to results page
